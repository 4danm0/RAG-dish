# 👨‍🍳 VK RAG Recipe Searcher: Умный Шеф

Продуманная поисково-генеративная система (**RAG**) для любителей вкусно поесть и не тратить часы на пролистывание пабликов. Проект разработан в рамках практического кейса от VK.

Приложение выкачивает кулинарные посты из ВКонтакте, очищает их от рекламы и спама, превращает тексты в векторы и складывает в локальную базу данных. Когда вы спрашиваете: *«Что приготовить из курицы и кабачка?»*, система находит подходящие рецепты и отдаёт их повару, который составляет для вас идеальную пошаговую инструкцию и список покупок.

---

## 🚀 Ключевые Фичи

*   **Анти-спам парсинг VK:** Интеллектуальная фильтрация постов. Никаких кликбейтов, ссылок на сторонние каналы, рекламы или постов в духе «Читать продолжение в источнике». Только чистый кулинарный контент.
*   **Автономная база данных:** Хранение векторов происходит локально в папке `qdrant_db_vk` прямо внутри проекта без необходимости развертывания тяжелых серверов БД.
*   **Гибридное облако ИИ:** 
    *   За генерацию высокоточных векторных смыслов (embeddings) отвечает модель `BAAI/bge-m3` от **Cloud.ru**.
    *   За кулинарную магию, структурирование и общение с пользователем — модель `deepseek-chat` через официальный **DeepSeek API**.
*   **Интуитивный UI:** Два режима работы («Простой поиск ссылок» и «Общение с ИИ-Шефом») в веб-интерфейсе на **Streamlit**.

## 🏗 Архитектура и Data Flow

```text
[ Паблики VK ] ──(VK API)──> [ parser.py: Очистка & Фильтрация ]
                                         │
                                   (Чистый текст)
                                         ▼
[ Локальный Qdrant ] <──(Запись)── [ indexer.py: Cloud.ru Embeddings ]
       │
 (Векторный поиск)
       ▼
[ searcher.py ] ──(Топ-3 рецепта)──> [ DeepSeek API (LLM) ] ──> [ Streamlit UI ]

```

---

## 🛠 Стек технологий

* **Язык программирования:** Python 3.10+
* **Интерфейс:** Streamlit
* **Векторная база данных:** Qdrant (Local persistent mode)
* **Клиенты API:** OpenAI SDK (универсальный интерфейс для Cloud.ru и DeepSeek)
* **Сбор данных:** Requests (VK API v5.131)

## ⚙️ Инструкция по запуску

### 1. Клонирование и подготовка окружения

```bash
# Клонируем репозиторий
git clone [https://github.com/ваш-юзернейм/vk-rag-searcher.git](https://github.com/ваш-юзернейм/vk-rag-searcher.git)
cd vk-rag-searcher

# Создаем и активируем виртуальное окружение
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate

# Устанавливаем необходимые библиотеки
pip install -r requirements.txt

```

### 2. Настройка переменных среды

Создайте файл с именем `.env` в корневой папке проекта и добавьте туда ваши ключи:

```env
VK_ACCESS_TOKEN=your_vk_service_or_user_token
CLOUD_API_KEY=your_cloud_ru_api_key
DEEPSEEK_API_KEY=your_deepseek_api_key

```

### 3. Запуск приложения

```bash
streamlit run app.py

```

После запуска консоль выдаст ссылку. Локальный интерфейс автоматически откроется в вашем браузере по адресу `http://localhost:8501`.

## 🧑‍🍳 Руководство пользователя

1. **Наполнение базы рецептов:**
При первом запуске ваша база пуста. Перейдите во вкладку **«⚙️ Обновление базы»** и нажмите кнопку **«Спарсить паблики VK и обновить индекс»**. Начнется автоматический сбор постов, их очистка и векторизация. В корне проекта создастся папка `qdrant_db_vk`, где теперь будут надежно храниться все рецепты.
2. **Поиск и кулинарные эксперименты:**
Перейдите во вкладку **«🔍 Поиск и ИИ-Шеф»**. Напишите любой запрос (например: *«быстрый завтрак из яиц»*).
* Кнопка **«Обычный поиск»** найдет наиболее подходящие по смыслу посты и покажет прямые ссылки на них в VK.
* Кнопка **«Спросить Шефа»** подключит искусственный интеллект DeepSeek, который прочитает найденные посты, объединит их и выдаст вам готовый структурированный рецепт со списком ингредиентов.
